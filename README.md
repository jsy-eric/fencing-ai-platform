# ⚔️ 击剑AI智能体平台

一个集成了击剑AI智能体、YouTube视频播放和弹幕系统的智能平台，为用户提供专业的击剑知识问答、实时弹幕生成和比赛数据分析服务。

## 🚀 功能特性

### 🎬 视频播放系统
- **YouTube集成**: 支持YouTube视频链接解析和嵌入播放
- **智能解析**: 自动提取视频信息，支持多种YouTube链接格式
- **响应式设计**: 适配不同屏幕尺寸，提供最佳观看体验

### 🤖 AI智能体系统
- **击剑专家**: 专业的击剑知识库，涵盖规则、技术、历史等
- **智能问答**: 基于上下文的智能问答系统
- **实时分析**: 根据视频内容提供实时分析和建议

### 💬 弹幕系统
- **用户弹幕**: 支持用户发送个性化弹幕
- **AI弹幕**: 基于视频内容的智能弹幕生成
- **实时显示**: 流畅的弹幕动画和显示效果
- **智能分类**: 自动分类和管理弹幕内容

### 📊 FIE数据集成
- **比赛结果**: 获取国际击剑联合会最新比赛数据
- **排名信息**: 实时世界排名和积分数据
- **赛事预告**: 即将到来的比赛信息

## 🛠️ 技术架构

### 后端技术
- **Flask**: Python Web框架，提供RESTful API
- **模块化设计**: 清晰的代码结构和模块分离
- **异步处理**: 支持并发请求和实时数据更新

### 前端技术
- **现代Web技术**: HTML5, CSS3, JavaScript ES6+
- **响应式设计**: 使用CSS Grid和Flexbox布局
- **组件化架构**: 模块化的JavaScript代码结构

### AI技术
- **知识图谱**: 结构化的击剑知识库
- **自然语言处理**: 智能意图识别和回复生成
- **上下文理解**: 基于视频内容的智能分析

## 📦 安装部署

### 环境要求
- Python 3.8+
- pip 20.0+
- 现代浏览器（Chrome, Firefox, Safari, Edge）

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd fencing_ai_platform
```

2. **创建虚拟环境**
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **运行应用**
```bash
python app.py
```

5. **访问应用**
打开浏览器访问 `http://localhost:5000`

## 🔧 配置说明

### 环境变量
创建 `.env` 文件并配置以下变量：
```env
FLASK_ENV=development
```

### 自定义配置
在 `config.py` 中可以修改：
- 弹幕显示设置

- 缓存配置
- API端点

## 📱 使用指南

### 1. 观看视频
1. 在YouTube链接输入框中粘贴击剑比赛视频链接
2. 点击"加载视频"按钮
3. 视频将自动嵌入播放器

### 2. AI智能问答
1. 在右侧AI聊天框中输入问题
2. 支持询问击剑规则、技术、历史等问题
3. AI将根据视频内容提供专业回答

### 3. 弹幕互动
1. 在弹幕输入框中输入内容
2. 点击"发送"按钮发布弹幕
3. 使用"AI生成弹幕"获取智能弹幕
4. 开启"自动生成"模式持续获取弹幕

### 4. 查看比赛数据
1. 在FIE数据区域查看最新比赛结果
2. 点击"刷新数据"获取最新信息
3. 了解世界排名和赛事预告
